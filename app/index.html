Paste an Anthropic API key to enable live research and generation (model: claude-opus-4-8 with web search). The key is stored only in this browser's localStorage — use a disposable, spend-capped key on a machine you control.

Org blocks browser requests? (“CORS requests are not allowed for this Organization”) — run node app/proxy.js in a terminal; the app detects it and switches automatically. The proxy holds the key server-side, which is also the recommended demo setup.

Without a key, the factory runs in clearly-labelled simulated mode — the reliable emergency backup for the demo.

Campaign Factory

Describe the local or public-policy problem you want to change. Campaign Factory researches it, verifies what it can, and turns it into a campaign objective, power analysis, strategy, organising plan, and a set of practical campaign resources.

Don't know who decides? That's fine — try: “I want to change [problem] in [location]. The outcome I want is [desired change]. This matters because [reason]. Help me identify who has the power to make the decision and how a campaign could influence them.” Imperfect prompts are expected; the research stage fills gaps and says what it couldn't establish.

Optimised for UK local government, public bodies, transport, planning, environment, education, health, public services, constituency issues and consultations. The factory follows: problem → research → objective → decision-maker → power → pressure → strategy → tactics → organising → drafted resources.
