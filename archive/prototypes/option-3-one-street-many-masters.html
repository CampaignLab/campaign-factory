One street, many masters

What it is: one frozen, citation-bound evidence pack about one street; flip the organisation and opposed campaigns come out — all citing the same sources. How it uses an agent factory: frame agents run in parallel against a shared frozen evidence store, a drift-checker agent audits every output against the locked claims, and the orchestrator can repeat the trick for any street × any organisation on the register. This page walks the six rungs from a chat opinion to manufactured pluralism.

Why parallel isn't theatre here

Context isolation: each frame agent argues its brief without seeing the others — exactly how four real organisations would behave. The orchestrator's job is to keep them honest against one shared store, not to make them agree.

About this concept

User & problem

A staffer at each of four fictional organisations — the point is that it's the same tool. The audience does the deciding: what does “grassroots” mean when opposed campaigns are manufactured from one shared record?

Where the agents multiply — and why it isn't theatre

Frame agents genuinely need separate contexts: each argues its organisation's brief without seeing the others, exactly as four real organisations would. The drift checker is a separate evaluator agent (generator/checker separation). The factory rung generalises the orchestration across two axes — streets and masters — with the frozen evidence store as shared infrastructure.

What's real on stage
Livethe frame toggle and re-syntheses — the wow is genuinely live, and the smallest live surface of the four

Risks & tensions

Stages synthetic campaigning and evidentiary laundering directly. No geographic scale — the factory point lands via multiplicity; if the panel needs volume, use the hybrid ending (scale the party frame across wards). Kill criterion: if models converge instead of opposing in rehearsal, the concept dies.

Build complexity & 10-minute fit

Lowest build of the four: one cached pack, parallel frame calls, a diff. Highest reliability. UI mirrors shadcn (Awake template tokens); production would use shadcn/ui React components directly.
