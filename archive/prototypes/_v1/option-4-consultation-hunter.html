Research finding that makes this option real: there is no aggregator of UK local-authority consultations — verified. GOV.UK lists national ones (71 open today, keyless JSON API), and the three dominant council platforms are all machine-readable (Citizen Space: open JSON API · Go Vocal: open undocumented API · Commonplace: embedded JSON). The thing this demo shows off does not currently exist. That is simultaneously the pitch and the warning.

Honest assessment

Strengths: the most defensible live layer of any option — three keyless JSON APIs verified today, no scraping fragility in the critical path. The Brent “you missed it” beat is emotionally precise and 100% true. Fills a verified real gap (no consultation aggregator exists), so it's genuinely useful beyond the conference. The closing question — what is participation worth at machine speed — is arguably the sharpest of the four.
Weaknesses: weakest on “meaningful agent behaviour” — the impressive part is feeds and deadlines (deterministic plumbing); the agent only ranks and drafts. No power map, no strategy, no doorstep — less of Randi's wish-list on screen. “Consultation” is a lower-blood-pressure word than “election”; the first 3 minutes must work hard to make the audience feel the stakes. Factory reveal needs the honest † caveat or it overclaims.
